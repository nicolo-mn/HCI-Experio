# Nome progetto

Experio - Travel Made Personal

# Mid-Fi Prototype

Le schermate che abbiamo scelto di prototipare sono la schermata home e l’inserimento di un nuovo consiglio. Sono state scelte perché rappresentano il cuore dell’applicazione, la visualizzazione dal punto di vista di un local che vuole dare consigli ad altre persone.

Per mostrare un feedback l’esito dell’interazione abbiamo aggiunto una notifica in sovraimpressione alla schermata.
